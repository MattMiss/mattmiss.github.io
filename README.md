# mattmiss.github.io

This repository contains the static files for my portfolio website hosted on GitHub Pages.

- **Website URL:** [https://mattmiss.github.io](https://mattmiss.github.io)
- **Source Code Repository:** [gatsby-portfolio](https://github.com/mattmiss/gatsby-portfolio)

## Deployment Notes
This site is built using [Gatsby](https://www.gatsbyjs.com/) and deployed by copying the contents of the `public` folder to this repository.
